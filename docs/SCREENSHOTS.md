# Screenshot Guidelines

Place your screenshots in `docs/images/` with the following names:

| Filename | Description |
|----------|-------------|
| `web-app.png` | Main web application interface |
| `results-dashboard.png` | Parsing results dashboard |
| `layer-structure.png` | Layer tree/structure visualization |
| `json-output.png` | JSON output example |

## Recommended Dimensions

- **Width**: 800-1200px
- **Format**: PNG or WebP
- **Dark mode preferred** for consistency

## How to Capture

1. Run the web application: `cd web && ./start.sh`
2. Upload a sample AI file
3. Capture screenshots of key features
4. Save to `docs/images/`
